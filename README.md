# frontend-medduzen-internship

## Requirements

- Node.js
- npm

## Installation



#### 1. Clone the Repository:


```bash
  git clone https://github.com/Bubl10k/frontend-medduzen-internship.git
```

#### 2. Install dependencies:

```bash
  npm install
```
#### 3. Start the React Vite development server:

```bash
  npm run dev
```
